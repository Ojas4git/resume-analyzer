# AI Resume Analyzer 🤖

This project is a web-based AI application that analyzes resume skills and recommends suitable job roles using Natural Language Processing (NLP).

## Features

* Input resume skills
* NLP-based text processing using TF-IDF
* Cosine similarity for matching
* Top job role recommendations
* Interactive web app using Streamlit

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* NLP (TF-IDF)

## How It Works

1. Convert resume and job skills into numerical vectors
2. Compare similarity using cosine similarity
3. Recommend the most relevant job roles

## Run the App

```bash
streamlit run app.py
```
